Catch the Ghost

  • Originality: This game is random and deterministic at the same time. Never seen a ghost catching game like this one.
  • Technical Merit: Use of time based functions to make everything work in sync.
  • Prototype Postmortem: Learned how to work with time based functions and how multiple states work (but only used two states the others were causing catching problems)

  • High Concept: Catch the ghost is a platfomer game where the overall objective is for the player to catch the ghost

  • Mandated Variety:
    1. Input: Mouse (also added keyboard incase player does not have a mouse)

    3. Play Styles: Joker (the game can be annoying at times)
    4. Genere: Platformer
  • Prototype Goal: Phaser's time based functions
  • Player Experience Goals: My overall goal is for the player to have fun
  • Gameplay: The basic premise of the game is to move around and catch the ghost. At times (every 30 secs) a star appears and if the player collects it they can fly for 5 seconds Controls
    1. To move left use left click (or the left arrow key)
    2. To move right use the right click (or right arrow key
    3. To jump use the middle scroll click (or the up key)
    4. If star consumed left click on places where you want to go Example
  • Strategies: Be patient and after playing the game for sometime you will find easier ways to catch the ghost
  • Story/Setting/Premise: So you are a ghost hunter, you have heard that there is a ghost around the graveyard and you have been assigned the task to catch this ghost.
  • Target Audience: Anyone who has the time to play web browser games.
  • Play Time: ~ 5 to 10 mins
